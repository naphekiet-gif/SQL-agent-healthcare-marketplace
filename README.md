# SQL Agent for Healthcare Marketplace

This project implements a SQL agent tailored for the healthcare marketplace, integrating various functionalities including data sourcing, query execution, and go-to-market insights generation.

## Features
- **Agent Logic**: Core SQL agent functionality.
- **LLM Integration**: Abstraction for large language model interactions.
- **SQL Execution**: Efficient execution of SQL queries across different data sources.
- **Go-to-Market Insights**: Analyze market strategies based on SQL outputs.
- **Healthcare Connectors**: Specific connectors to healthcare data sources.
- **FastAPI Integration**: Expose functionalities as a REST API.

## Requirements
List of Python dependencies can be found in [requirements.txt](requirements.txt).

## Setup
1. Clone the repository.
2. Set up your environment using the .env.example file.
3. Install required packages using the command:
   ```
   pip install -r requirements.txt
   ```

## Usage
To use the SQL agent, refer to the example queries provided in the `examples` folder.
